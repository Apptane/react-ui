### Content Positioning

This component supports flexbox-based content positioning with more intuitive API. Use `orientation` to specify the direction of the child content. Use `horizontalAlignment` and `verticalAlignment` to align content. These properties work as their names imply regardless of the orientation selected.

export const PaneContentTemplate = (args) => (
  <div>
    <Pane width={120} height={120} background="default" padding={5} {...args}>
      <Pane width={20} height={20} background="contrast" margin={5} />
      <Pane width={20} height={20} background="contrast" margin={5} />
    </Pane>
    <Text size="small" color="muted" marginTop={4}>
      {args.horizontalAlignment} â€” {args.verticalAlignment}
    </Text>
  </div>
);

Use `margin` and `padding` to create spacing outside and inside the panel. You can also set individual values for individual sides via `marginTop|Right|Bottom|Left` and `paddingTop|Right|Bottom|Left` or using shortcut properties: `m|mt|mr|mb|ml` and `p|pt|pr|pb|pl`.

### Card

`Card` is a special version of `Pane` with default border and background. Use `interactive` property to enable visual response to mouse and keyboard focus.
