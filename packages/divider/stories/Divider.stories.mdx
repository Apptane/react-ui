### Interactivity

Setting `onChange` property to a valid callback enables interactive behavior. The divider can be used as an expansion toggle.

  <Story name="Interactivity">

      {({ state, setState }) => (

          HEADING
        </Divider>
      )}
    </StateManager>
  </Story>
