## Appearances

The banner default appearance is the borderless panel. It supports custom actions panel that can be located to the left or to the bottom of the main content.

export const BannerActions = () => (
  <Pane orientation="horizontal" horizontalAlignment="right">
    <Button appearance="minimal">Cancel</Button>
    <Button marginLeft={8}>OK</Button>
  </Pane>
);

<Canvas>
  <Story name="Actions — Horizontal">
    <Banner actions={<BannerActions />}>
      Please read this message and take the prescribed action for the desired efffect
    </Banner>
    <Banner intent="neutral" actions={<BannerActions />} marginTop={16} width={450}>
      Please read this message and take the prescribed action for the desired efffect
    </Banner>
  </Story>
  <Story name="Actions — Vertical">
    <Banner actions={<BannerActions />} orientation="vertical">

  </Story>
</Canvas>

### Card

Setting `card` property renders the banner in card with intent-colored border. Padding can also be customized.

<Canvas>
  <Story name="Card">
    <Banner intent="warning" card padding={16} maxWidth={560}>
      You do not have an IT environment yet — one will be automatically provisioned for you in the background as you
      accept this invitation
    </Banner>
  </Story>
</Canvas>
