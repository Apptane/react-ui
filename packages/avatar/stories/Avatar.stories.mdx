## Appearances

### Gravatar

Set `type = gravatar` to use automatically fetch avatar from Gravatar service by `email`.
Alternative `image` can be specified as an URI to display.

<Canvas>
  <Story
    name="Gravatar"
    parameters={{
      backgrounds: {
        default: "dark",
      },
    }}>
    <Avatar type="gravatar" email="oleg.grodzevich@gmail.com" inline />
    <Avatar
      type="gravatar"
      email="oleg.grodzevich@gmail.com"
      inline
      marginLeft={16}
      size="large"
      tooltip="Oleg Grodzevich"
    />
    <Avatar type="gravatar" email="oleg.grodzevich@gmail.com" inline marginLeft={16} square />
    <Avatar
      type="gravatar"
      email="oleg.grodzevich@gmail.com"
      inline
      marginLeft={16}
      size="large"
      tooltip="Oleg Grodzevich"
      square
    />
  </Story>
</Canvas>

### Colors

Colors can be customized by setting `color` to one of the palette's hues or to a valid CSS color.
This mode applies only to avatars displaying initials.

## Group

Use `AvatarGroup` to display a stacked row of avatars.

      <Avatar name="Alice" color="green" />
      <Avatar name="Bob" color="blue" />
    </AvatarGroup>
    <AvatarGroup size="medium" marginTop={16}>
      <Avatar type="gravatar" email="oleg.grodzevich@gmail.com" />
      <Avatar name="Oleg Grodzevich" />
      <Avatar name="Iryna Zolotarenko" color="red" />
      <Avatar name="Alice" color="green" />
      <Avatar name="Bob" color="blue" />
    </AvatarGroup>
    <AvatarGroup size="large" marginTop={16}>
      <Avatar type="gravatar" email="oleg.grodzevich@gmail.com" />

      <Avatar name="Iryna Zolotarenko" color="red" />

The stack size can be controlled via `maxCount` or `maxWidth` properties. The first one restricts the stack to the maximum number of items and the second one indicates the maximum width to fit into. Overflow mode is enabled automatically.
